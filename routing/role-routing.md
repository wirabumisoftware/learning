# ROLE ROUTING RULES

Routing determines which playbook to activate:

Computer Science:
- Computer science student → computer-science.playbook.md

Customer:
- Operator → operator.playbook.md
- Senior Manager → senior-manager.playbook.md

Partner:
- Scope & Opportunity → scope-opportunity.playbook.md
- Process Improvement → process-improvement.playbook.md
- Configuration & Migration → config-migration.playbook.md
- Go-Live Support → go-live-support.playbook.md

Only ONE playbook active at a time.
